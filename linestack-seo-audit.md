# LineStack ¨C Page-Level SEO & Copy Audit
## Block-by-Block Improvement Plan

> Product Name: LineStack  
> Slogan / Functional Descriptor: Vertical Movie Screenshot Stitcher

---

## 0. Overall Goal (Context)

This document provides a block-by-block copy and SEO improvement plan for the LineStack landing page.

Core objective:
- Preserve **LineStack** as a long-term brand
- Use **Vertical Movie Screenshot Stitcher** as the primary SEO and user-understanding anchor
- Reduce ambiguity
- Increase semantic clarity for search engines
- Lower cognitive cost for first-time users

---

## 1. Header / Top Navigation Block

### Current State
- LineStack is presented as the main visible name
- Functional meaning is not immediately clear from the header alone

### Problem
- Brand-only exposure gives no semantic signal to new users or search engines
- Users must scroll or infer usage

### Improvement Direction
- Keep LineStack visually dominant
- Introduce the slogan **near the brand**, not competing with it
- Ensure the slogan is readable without scrolling

### Key Principle
Brand identifies.
Slogan explains.

---

  - Why is it different?

### Improvement Direction
Hero block should answer three questions in order:

1. What is the product?  
   ¡ú LineStack

   ¡ú Vertical Movie Screenshot Stitcher
3. What makes it special?  
   ¡ú Vertical continuity + subtitle preservation

### Structural Recommendation
- LineStack as the identity anchor
- Vertical Movie Screenshot Stitcher as the semantic anchor
- One supporting sentence explaining vertical + subtitles

## 3. First Descriptive Paragraph (SEO Critical)

### Current State
- Either very short or overly tool-focused
- Lacks semantic density

### Problem
- Search engines need a clear textual definition
- Current content does not strongly establish topic authority

### Improvement Direction
This block should:
- Clearly define what a ¡°Vertical Movie Screenshot Stitcher¡± is
- Mention movies, TV shows, screenshots, vertical layout, subtitles
- Explain how this differs from generic image collages

### Key Rule
This paragraph is written for **search engines first**, humans second.

---

## 4. Core Feature Block

### Current State
- Features may exist but are generic or under-explained

### Problem
- Generic features do not differentiate
- Missing opportunity to reinforce keywords semantically

### Improvement Direction
Each feature should:
- Reinforce one core concept:
  - Vertical continuity
  - Subtitle preservation
  - Cinematic rhythm
- Avoid vague UI language
- Avoid marketing fluff

### Semantic Focus
Do not describe buttons.
Describe outcomes.

---

## 5. Differentiation Block (Very Important)

### Current State
- LineStack may feel similar to:
  - Image collage tools
  - Screenshot stitchers
  - Meme generators

### Problem
- Without explicit differentiation, users misclassify the tool
- Search engines group you with the wrong competitors

### Improvement Direction
Explicitly state what LineStack is NOT:
- Not a chat screenshot tool
- Not a meme generator
- Not a generic collage maker

Then state what it IS:
- A tool for linear cinematic storytelling
- Focused on subtitles and sequence continuity

---

## 6. Use Case Block

### Current State
- Either missing or too abstract

### Problem
- Lacks real-world grounding
- Weak long-tail keyword coverage

### Improvement Direction
Use role-based scenarios:
- Film reviewers
- Screenwriters
- Movie lovers
- Social media storytellers

Each use case should answer:
- Who uses it?
- Why vertical matters?
- Why subtitles matter?

This block significantly boosts semantic richness.

## 7. FAQ Block (SEO Weapon)

### Current State
- Either minimal or written like customer support

### Problem
- Missed opportunity for long-tail search traffic

### Improvement Direction
FAQ questions should:
- Be written as real search queries
- Contain keywords naturally
- Reinforce ¡°Vertical Movie Screenshot Stitcher¡±

- How is this different from collage tools?

Answers should be explanatory, not short.
